# NFTs

Once you have minted an NFT through a Minter Checkout, you can access it through various queries in order to retrieve information about the NFT. Since NFTs are immutable, you are not able to make any changes to them. You can only query them in order to retrieve information about the NFT, such as its unique identifier, metadata, and any associated content

## MinterNft Object

- `id`: The unique identifier for the event.
- `project`: The project that the NFT belongs to.
- `owner`: The address of the owner of the NFT.
- `mint_tx`: The transaction hash of the minting transaction.
- `network`: The blockchain network that the NFT is on.
- `token_id`: The unique identifier for the NFT on the blockchain.
- `owner_synced_at`: The timestamp of when the NFT's ownership was synced with the blockchain.
- `mint_request_id`: The unique identifier for the mint request.
