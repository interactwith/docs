# Minter Checkouts

Minter Checkout is the entry for NFTs. It is used for minting project's NFTs.

### Create a checkout

<Tabs>
<TabItem value="js" label="JavaScript">
<JsExample>

```
const checkout = await interactwith.minter.minter_checkouts.create({
  name: 'Checkout 1',
  description: 'Checkout 1 description',
  price: 100,
  currency: 'USD',
  project: 'PROJECT_ID',
  metadata: {
    name: 'Checkout 1',
    description: 'Checkout 1 description',
    image: 'https://example.com/image.png',
  },
})
```

</JsExample>
</TabItem>
<TabItem value="gql" label="GraphQL">
<GqlExample resource="DassetsCheckoutSession" method="create"/>
</TabItem>

</Tabs>

<TabItem value="js" label="JavaScript">
