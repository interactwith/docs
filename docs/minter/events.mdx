# Events List

Using [Event Emitters](/getting-started/event-emitters) you can listen for events that are triggered by the [API](/getting-started/api).

### Minter.NftTransferred

```json
{
  "type": "Minter.NftTransferred",
  "data": {
    "token_id": "1",
    "from": "Mx0000000000000000000000000000000000000000",
    "to": "Mx0000000000000000000000000000000000000000"
  }
}
```

### Minter.NftBurned

```json
{
  "type": "Minter.NftBurned",
  "data": {
    "token_id": "1",
    "owner": "Mx0000000000000000000000000000000000000000"
  }
}
```

### Minter.NftMinted

```json
{
  "type": "Minter.NftMinted",
  "data": {
    "token_id": "1",
    "owner": "Mx0000000000000000000000000000000000000000"
  }
}
```
